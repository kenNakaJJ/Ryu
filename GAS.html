仕様書 / README.md テンプレート

1. 概要 (Overview)

毎朝9時に売上速報データを集計し、関係者へサマリーメールを送信するスクリプトです。

2. 目的・解決する課題 (Purpose)

これまで手動で行っていた日次レポート作成業務を自動化し、〇〇時間の工数削減と、集計ミスを防止することを目的とします。

3. 主な機能 (Features)

  • 〇〇スプレッドシートからのデータ取得
  • データの集計・整形
  • メール本文の自動生成
  • 指定アドレスへのメール送信

4. 処理フロー (Workflow)

  1. 1. 時間主導型トリガーにより main 関数が起動
  2. 2. 〇〇スプレッドシートからデータを取得
  3. 3. データを整形・集計
  4. 4. エラーがなければメールを送信
  5. 5. ログ用スプレッドシートに実行結果を記録して終了

5. 関連リソース (Related Resources)

3. スクリプトプロパティの設定 (Script Properties)

本スクリプトは、変更される可能性のある設定値をスクリプトプロパティで管理しています。
ファイル > プロジェクトの設定 > スクリプト プロパティ から以下の値を設定してください。

キー 説明 設定例
SPREADSHEET_ID 処理対象のスプレッドシートID 12345abcdefg_hijklmnOPQRSTU
ADMIN_EMAIL エラー通知を受け取る管理者のメールアドレス admin@example.com
API_KEY 使用する外部サービスのAPIキー （ここに直接書かず、プロパティに設定する）
注意: APIキーやパスワードなどの機密情報を、このドキュメント内に直接記述しないでください。

4. トリガーの設定 (Triggers)

編集 > 現在のプロジェクトのトリガー から、以下の設定でトリガーを新規作成してください。

  • 実行する関数を選択: main
  • イベントのソースを選択: 時間主導型
  • 時間ベースのタイマーの種類を選択: 日付ベースのタイマー
  • 時刻を選択: 午前 8 時～9 時

利用者マニュアル テンプレート

1. このツールでできること

このツールを使うと、スプレッドシートの「カスタムメニュー」からワンクリックで、最新の請求書PDFを作成し、指定のフォルダに保存できます。

2. 使い方 (How to Use)

  1. 1. 請求書データが入力されたスプレッドシートを開きます。
  2. 2. 上部メニューの [カスタムメニュー] > [請求書作成] をクリックします。
  3. 3. 「処理を実行しますか？」というダイアログが表示されたら「はい」をクリックします。
  4. 4. 「完了しました」と表示されたら、〇〇フォルダにPDFが作成されています。

3. 注意事項 (Notes)

  • 一度に処理できるデータは100行までです。
  • 処理には数分かかる場合があります。完了メッセージが表示されるまでブラウザを閉じないでください。

4. エラーかな？と思ったら (Troubleshooting)

  • Q. メニューが表示されません。

    A. ページを再読み込み（F5キー）してみてください。それでも表示されない場合は、管理者にお知らせください。

  • Q. 「エラーが発生しました」と表示されます。

    A. 時間を置いて再度試してみてください。何度も発生する場合は、エラーメッセージを控えて管理者にご連絡ください。
